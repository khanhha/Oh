building steps:

1. set up template 3rd parties: 	
	- \3rdParty\libigl\include
	- \3rdParty\vcglib
	- \3rdParty\eigen

2. set up opencv
	- include dir: opencv_lib\opencv\build\include
	- lib dir: opencv_lib\opencv\build\x64\vc14\lib
	- specific opencv libs: opencv_world341.lib

3. build 

4. how how use
	- change path in DecalPainter_Test.cpp to the correct file in your local machine
	- run DecalPainter_Test
